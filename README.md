# spring-booster

Pluggable toolkit for CURD

## Modules

- authentication
  - jwt
  - rbac authentication
- core
  - common spring boot setup
  - spring boot utils
- dao data access layer
  - jpa with snowflake
- file operation
  - s3: minio/others
  - local
- web
- office
  - excel
  - word
- helper library
  - mutiny integration
- tools
  - annotation code generation
  - cli template code generation
    - generate from database schema
  - build tool plugins
    - gradle 
    - maven
